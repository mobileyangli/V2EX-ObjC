
## V2EX-ObjC ##


ObjC实现的第三方V2EX客户端，本着实验的目的来做。项目按照产品的方向实验，实验目标：Scrum管理和使用开发的最佳实践。

涉及资源：

 1. 项目管理看板，[Trellor](https://trello.com/b/9uhdhn4m/v2ex-objc)
